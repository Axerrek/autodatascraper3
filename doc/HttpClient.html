    class HttpClient

    Klasa do wykonywania żądań HTTP i pobierania stron internetowych.

    Klasa ‘HttpClient` używa biblioteki `HTTParty` do pobierania zawartości stron internetowych. Pozwala na wykonanie żądań HTTP GET z domyślnymi nagłówkami.

    fetch_page(path) click to toggle source

    Pobiera zawartość strony HTML na podstawie podanej ścieżki.

    Metoda wysyła żądanie HTTP GET do zbudowanego URL-a i zwraca ciało odpowiedzi jako tekst.

    path - Ścieżka, która zostanie dołączona do podstawowego URL-a, aby utworzyć pełny URL. @return [String] Ciało odpowiedzi jako tekst, które zawiera zawartość strony HTML.

    # File classes/http_client.rb, line 24
def fetch_page(path)
  response = HTTParty.get("#{@base_url}#{path}", {
    headers: {
      "User-Agent" => "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/109.0.0.0 Safari/537.36"
    }
  })
  response.body
end
